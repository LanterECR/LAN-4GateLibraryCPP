◆ CallbackNotificationType

enum class Lanter::Manager::ILan4Gate::CallbackNotificationType
strong

Способ вызова колбеков Объединение Async | Sync будет интерпретировано, как Sync.

Элементы перечислений
Async 

Все колбеки вызываются из того же потока, в котором работает метод doLan4Gate.

Sync 

Все колбеки вызываются из нового потока
